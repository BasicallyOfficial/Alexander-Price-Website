Glitch Art

This project is an example of what is known as "Glitch Art." It is done by taking the file of an image into audacity, and messing with the source code to create interesting art within an existing image by breaking down into something new. I used a picture of a hotdog, a photo of a man giving a thumbs up, and a mountainous landscape.
