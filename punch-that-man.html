Punch That Man!

This project was my return to Javascript in order to create an interactive, browser-based artpiece. Inspired by the repeated stock sound effects commonly used in videogames whenever a character gets hurt, I decided to create a punchable man that exclaims a randomly selected sound effect using curated assets from freesound.org. In addition, the man's face reacts according to the intensity and volume of the sound being played, and a volume slider has been added to allow for more user-control. My full Artist Statement is accessible via the Punch That Man! website. In order to access the website and try it out yourself, just click on the title above!
