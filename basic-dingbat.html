A Very Basic Font

This is a font that I created with various symbols of my choosing using Adobe Illustrator and Fontlab 7. The picture above depicts most of the symbols in the font, which you can download for yourself with the link below. To use the font, open the downloaded file and press the install button at the top. This will allow you to use my custom dingbats font in most Windows 10 software such as Word, Notepad, and Wordpad. (I recommend a large font size for the best results) For a full deconstruction of my ideas, process, and challenges you can download the PDF of my full artist statement at the bottom of the page.
